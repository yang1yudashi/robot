# 学习路线入口

这里按章节组织学习路线。每天学习时先看第 1 章；第 2、3 章只在需要背景时阅读。

## 第 1 章：当前两周学习任务

文件：[加速学习计划](./accelerated-plan.md)

这一章是每日学习任务的唯一入口，回答的是：

- 今天学哪个脚本？
- 今天要运行什么实验？
- 今天必须理解哪些概念？
- 今天做到什么程度算完成？
- 本周什么时候进入下一个主题？

你说“开始今天学习”时，就按这一章安排。

## 第 2 章：Linker Hand + BODex 项目路线

文件：[Linker Hand + BODex 项目背景路线](./reference/linkerhand-bodex.md)

这一章解释当前项目为什么这样学，回答的是：

- Linker Hand、BODex、MuJoCo、cuRobo 分别负责什么？
- 为什么要先跑通 MuJoCo，再进入 BODex？
- 后面如何整理 URDF、joint、link 和 joint_map？
- BODex 输出结果之后，怎么送进仿真验证？
- 真机执行前需要哪些安全准备？

这一章是项目背景，不作为当天作业。

## 第 3 章：具身智能长期路线

文件：[具身智能长期学习路线](./reference/embodied-ai-roadmap.md)

这一章解释更大的学习方向，回答的是：

- 具身智能是什么？
- 机器人、仿真、控制、感知、学习、VLA 之间是什么关系？
- MuJoCo + Linker Hand + BODex 在整个具身智能路线里属于哪一段？
- 后续什么时候学 ROS 2、视觉、点云、模仿学习和强化学习？

这一章只用于了解长期方向，不影响当天任务。

## 使用规则

- 每日任务只来自第 1 章。
- 第 2 章用于理解当前项目，不抢第 1 章的优先级。
- 第 3 章用于理解长期方向，不安排当天作业。
- 如果章节之间看起来冲突，以第 1 章为准。
